#TODO cli program

A terminal-based todo program written in Go, usable directly in your terminal using raw keyboard input.

## Requirements

- Go 1.21 or higher
- A Unix-based terminal (Linux or macOS)

## Installation

Clone the repository and install dependencies:

```bash
git https://github.com/rashmithmr153/Todo.git
cd Todo
go install
```

## Running the Program
```bash
todo  -for tui
todo [option][value] for cli
```
